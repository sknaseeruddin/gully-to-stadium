# 🏏 Gully to Stadium

A live cricket scores and upcoming matches web app built using HTML, CSS, and JavaScript.

## Features
- Live match scores
- Upcoming matches
- Search by team
- Filter by match type
- Filter by series
- Dark/Light mode
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- CricAPI

## Current Status
This project is under development. Some advanced features like full scorecard, ball-by-ball commentary, and API optimization are still being improved.
