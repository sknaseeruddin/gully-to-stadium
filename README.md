# 🏏 Gully to Stadium

🚀 A live cricket scores and upcoming matches web application built using HTML, CSS, and JavaScript.

---

## 🔥 Features
- 📡 Live cricket scores  
- 📅 Upcoming matches  
- 🔍 Search teams  
- 🎯 Filter by match type  
- 📊 Filter by series  
- 🌙 Dark / Light mode  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- CricAPI  

---

## ⚠️ Note
This project uses a free API.  
Sometimes live data may not load due to API request limits.
